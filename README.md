## Title
Weight Lifting GUI App

## Description
Takes in a weight in lbs and the number of reps. Then returns a calculated max along with the calculated percentages of the largest calculated max.

For example: say you benched press 255 lb for 10 reps. The app would take those numbers and use the 3 most popular  
max calculation equations and calculate your estimated max bench press.
